## 约定

- 每次分阶段落地 / 重要变更产出一份说明，按 `主线X-PhaseN-简述.md` 命名
- 内容包括：目标、改动范围、按文件的详细变更、向后兼容性证明、验收测试、回滚方法、给不同角色的速查
- **不在此记录日常小改动**（那些走 git commit 说明即可）
- 本目录下的文档是"定版说明"，代码有后续调整时 → 更新 `数据源改造方案.md` 等主文档，不改这里
