# SQLBot-cx 数据中台集成文档

本目录是 SQLBot-cx 接入数据中台（统一认证 + 统一数据源 + 统一权限）的**现行**文档集合。

## 阅读顺序

| # | 文档 | 用途 |
|---|---|---|
| 1 | [`集成总览.md`](./集成总览.md) | 集成全景：两条主线、当前进度、关键决策 |
| 2 | [`配置手册-环境变量.md`](./配置手册-环境变量.md) | `.env` 每个字段的作用、取值含义、故障排查信号 |
| 3 | [`认证体系总览.md`](./认证体系总览.md) | 认证改造（主线一）全景与代码定位 |
| 4 | [`数据源改造方案.md`](./数据源改造方案.md) | 数据源改造（主线二）方案与分阶段落地 |
| 5 | [`变更/`](./变更/) | 分阶段落地的详细改动说明（团队宣讲 / review 用） |

## 与 `archive/` 目录的关系

`archive/` 下是认证改造期间产出的过程文档，已归档不再维护。历史决策背景可回看，技术事实以本目录和代码为准。

## 更新约定

- **代码改动 = 文档同步更新**。认证/数据源相关代码变更时，同步改动 `认证体系总览.md` / `数据源改造方案.md`。
- **配置新增 = `配置手册-环境变量.md` 必须同步**。新增的 `.env` 字段需立刻补文档，否则相当于未提交。
- **敏感值全部用占位符**：不写真实 token、账号、密码、租户 ID、内部 IP 直接复制进文档。
